# 🕵️‍♂️ Sherlock & Watson: The Case of the Pixelated Monsters

Bem-vindo ao submundo de Londres! Este é um jogo de plataforma e ação **cooperativo para 2 jogadores** desenvolvido no **Greenfoot**. Aqui, a dedução dá lugar à sobrevivência enquanto Sherlock Holmes e o Dr. Watson enfrentam hordas de monstros.

---

## 🎮 Sobre o Jogo

Nesta aventura, o trabalho de equipa é fundamental. Enquanto um jogador foca no combate, o outro pode ajudar a resolver puzzles ou cobrir a retaguarda. O jogo desafia a vossa coordenação em saltos precisos, lógica em enigmas e resistência em combates intensos.

### **Funcionalidades Principais:**

* **👥 Modo 2 Jogadores:** Joga com um amigo no mesmo teclado (Sherlock & Watson).
* **🧱 Plataformas:** Atravessa cenários vitorianos perigosos com saltos precisos.
* **🧩 Puzzles:** Pequenos enigmas que exigem a interação de ambos para avançar.
* **👾 Combate a Monstros:** Enfrenta criaturas bizarras com diferentes padrões de ataque.
* **🌀 O Labirinto:** Um teste à vossa orientação e comunicação antes do desafio final.
* **⚔️ Modo Waves (Ondas):** Sobrevivam juntos a hordas de inimigos num mapa de arena.
* **👹 Boss Final:** Uma batalha épica que exigirá o máximo da vossa estratégia.

---

## 🚀 Como Jogar

1.  **Pré-requisitos:** Certifica-te de que tens o [Greenfoot](https://www.greenfoot.org/download) instalado.
2.  **Download:** Clona este repositório ou faz download do ficheiro `.zip`.
3.  **Abrir:** Inicia o Greenfoot e abre o ficheiro `project.greenfoot`.
4.  **Executar:** Clica em **Run** e chama o teu parceiro de crime!

### **Controlos:**

| Ação | Jogador 1 (Sherlock) | Jogador 2 (Watson) |
| :--- | :--- | :--- |
| **Movimento** | Teclas `W`, `A`, `S`, `D` | Setas `↑`, `←`, `↓`, `→` |
| **Ataque** | Tecla `F` | Tecla `L` |
| **Salto** | Tecla `E` | Seta `K` |

## 🛠️ Tecnologias Utilizadas

* **Motor:** [Greenfoot](https://www.greenfoot.org/)
* **Linguagem:** Java
* **Multijogador:** Lógica de inputs simultâneos para dois atores independentes.

---

## 📁 Estrutura do Projeto

* **Actors:** Classes específicas para `Sherlock` e `Watson`, além de classes para `Inimigos`, `Projéteis` e `Obstáculos`.
* **Worlds:** Níveis estruturados para progressão: Plataformas -> Puzzles -> Labirinto -> Waves -> Boss.
